Trigger alert on scroll down

Set popup_scroll_trigger argument's value as true and pass the scrollable container's id/class as popup_scrollable_container's value.

Here is a preview of used arguments on the popup button. The popup_scroll_trigger value is set to true and element with the class 'elements-container' is passed as 'popup_scrollable_container'. Now on scrolling down on the 'elements-container' div, it will show alert but not on scrolling to top:

        
            <button onclick="initiate_popup({
                'target': '#scroll-popup',
                'overlay_color': 'rgba(0,0,0,0.65)',
                'popup_scroll_trigger': true,
                'popup_scrollable_container': 
                '.elements-container',
            });
            ">
                Show/Hide trigger alert
            </button>
